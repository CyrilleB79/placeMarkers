Este complemento sirve para guardar y buscar textos específicos y marcas en páginas web, o en documentos mostrados en el modo navegación de NVDA. El complemento guarda las búsquedas y marcas en ficheros de texto y con extensión pickle. El nombre de estos archivos se basa en el título del documento actual. Al ejecutar el comando de búsqueda específica, NVDA abrirá un diálogo que mostrará el texto correspondiente al documento actual, si se ha guardado algún texto para este documento.

placeMarkers se basa en SpecificSearch and Bookmark&Search, creado por la misma autora. Debes desinstalarlos para usar el complemento PlaceMarkers, puesto que tienen combinación de teclas y funciones en común.

Funciones:

Atajos de teclado:

control+shift+NVDA+s
Abre un diálogo para guardar un texto de búsqueda asociado al documento actual. Por defecto muestra el texto guardado para este documento. Si se borra el texto que aparece en el diálogo, se eliminará el correspondiente fichero de búsqueda.
control+shift+NVDA+f
Si existe un fichero de búsqueda para el documento actual, abre un diálogo que mostrará el texto guardado. Al aceptar, NVDA buscará el texto que se indica en el cuadro de edición. Si no se encuentra un fichero para el documento, NVDA avisa con un mensaje sin abrir el diálogo de búsqueda específica.
control+shift+NVDA+k
Guarda la posición actual del cursor como una marca.
control+shift+NVDA+supr
Borra la marca correspondiente a esta posición.
control+shift+k
Se mueve a la marca siguiente.
shift+NVDA+k
Se mueve a la marca anterior.
NVDA+k
Anuncia el nombre del archivo, sin la extensión, en el que podrían estar guardadas las marcas de posición o de las búsquedas específicas.

Place markers Submenú (NVDA+N)

En el menú Place markers, integrado en el submenú Preferencias, se puede acceder a:

  • La carpeta de búsqueda específica; muestra una carpeta con búsquedas específicas, que has guardado.
  • La carpeta de marcas; muestra la carpeta des marcas guardada previamente.
  • Copia la carpeta Marcas; te permite de guardar una copia de la carpeta Marcas.
  • Restaurar Marcas; puedes restaurar la carpeta Marcas desde una copia guardada previamente.
  • Guía del usuario, Si está disponible en el idioma seleccionado, de lo contrario por defecto se abre la guía en Inglés.

Nota: La posición de la marca se basa en el número de caracteres; en la páginas con contenido dinámico, es mejores utilizar la búsqueda específica y no las marcas para guardar una posición más precisa.
