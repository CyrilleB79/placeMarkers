  2. Informação da página

Este complemento serve para guardar e procurar textos específicos e marcadores em páginas web, ou em documentos mostrados no modo de navegação do NVDA. O complemento guarda as buscas e marcadores em ficheiros de texto e com extensão pickle. O nome destes arquivos baseia-se no título do documento actual. Ao executar o comando de busca específica, o NVDA abrirá um diálogo que mostrará o texto correspondente ao documento actual, se se tiver guardado algun texto para este documento.

placeMarkers baseia-se no SpecificSearch and Bookmark&Search, criado pela mesma autora. Deves desinstalá-los para usar o complemento PlaceMarkers, uma vez que têm comandos e funções em comum.

Comandos:

Comandos de teclado:

control+shift+NVDA+s
Abre um diálogo para guardar um texto de procura associado ao documento actual. Por padrão, mostra o texto guardado para este documento. Se se apaga o texto que aparece no diálogo, elimina-se o correspondente ficheiro de busca.
control+shift+NVDA+f
Se existir um ficheiro de busca para o documento actual, abrirá um diálogo que mostrará o texto guardado. Ao aceitar, o NVDA procurará o texto que se indica na caixa de edição. Se não se encontrar um ficheiro para o documento, o NVDA avisará com uma mensagem, sem abrir o diálogo de busca específica.
control+shift+NVDA+k
Guarda a posição actual do cursor como um marcador.
control+shift+NVDA+supr
Apaga o marcador correspondente a esta posição.
control+shift+k
Move-se para o marcador seguinte.
shift+NVDA+k
Move-se para o marcador anterior.
NVDA+k
Anuncia o nome do arquivo, sem a extensão, onde poderiam estar guardados os marcadores de posição ou de buscas específicas.

Place markers Submenu (NVDA+N)

No menu Place markers, integrado no submenu Preferências, pode aceder a:

  • Pasta de busca específica; mostra uma pasta com buscas específicas, que foi anteriormente guardada.
  • Pasta de marcadores; mostra a pasta de marcadores, guardada anteriormente.
  • Copiar pasta de Marcadores; permite guardar uma cópia da pasta de Marcadores.
  • Restaurar Marcadores; podes restaurar a pasta Marcadores a partir de um backup guardado previamente.
  • Guia do usuário, Se estiver disponível no idioma seleccionado; caso contrário, por padrão, abre oguia em Inglês.

Nota: A posição de marcadores baseia-se no número de caracteres; em páginas com conteúdo dinâmico, é melhor usar a busca específica e não os marcadores para guardar uma posição mais precisa.
