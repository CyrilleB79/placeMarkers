Marqueurs&Recherche 2.0

Ce module permet de rechercher et sauvegarder Des marqueurs ou des chaînes de caractères spécifiques dans des pages web ou des documents sous NVDA en mode navigation. Les textes et les marqueurs sont sauvegardés dans des fichiers. Le nom de ces fichiers est basé sur le titre du document en cours. Quand on exécute la commande de recherche spécifique, NVDA affiche un dialogue contenant le texte spécifique sauvegardé s'il existe pour ce document.

Commandes

Clavier

control+maj+NVDA+s
Ouvre un dialogue permettant d'enregistre une chaîne de caractère que vous désirez rechercher dans le document. Par défaut, Le texte précédemment sauvé pour ce document est affiché. Si on efface le texte, le fichier contenant le texte sauvegardé sera également effacé.
control+maj+NVDA+f
Si le document courant possède un fichier de recherche spécifique, un dialogue s'ouvre affichant le texte précédemment recherché. Quand on appuie sur OK, NVDA recherche le texte présent dans la zone d'édition du dialogue. S'il n'y a pas de fichier texte pour ce document, NVDA l'annonce sans ouvrir de dialogue.
control+maj+NVDA+k
Sauvegarde la position courante comme marqueur.
control+maj+NVDA+suppr
Efface le marqueur correspondant à cette position.
control+maj+k
Aller au marqueur suivant.
maj+NVDA+k
Aller au marqueur précédent.
NVDA+k
Annonce le nom du fichier, sans extension, dans lequel pourraient être sauvegardés des marqueurs de position ou des recherches spécifiques.

Sous-menu des Marqueurs de Position (NVDA+N)

Dans le sous-menu de recherche spécifique, dans le menu Préférences, vous trouverez :

  • Le dossier de recherche spécifique.
  • Le dossier des marqueurs.
  • Le fichier de documentation, si possible dans votre langue ou en Anglais par défaut.
